Interactive Loyc

interactive or "interpreted" Loyc compiles a statement typed by the user and then executes it if it is executable. Declarations go into a special class called InteractiveStmts, and executable statements are run in the context of a method called ImmediateStmts. Extensions that want to be available in interactive mode need simply to add their syntax [TODO] to the add-in tree in /Stmts/Categories/Interactive, in addition to other locations used for normal compilation.
